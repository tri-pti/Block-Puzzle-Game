# Block Puzzle Game

A simple and engaging block puzzle game built using **HTML**, **CSS**, and **JavaScript**. Players strategically place different block shapes on an 8×8 grid, clear completed rows and columns, and aim to achieve the highest possible score.

## Features

* 8×8 interactive puzzle board
* Multiple block shapes
* Click-to-select and click-to-place gameplay
* Placement preview with color indication
* Row and column clearing
* Score and best score tracking
* Best score saved using Local Storage
* Restart game option
* Responsive and lightweight design
* Built with pure HTML, CSS, and JavaScript

## Run It

Simply open **`index.html`** in any modern web browser.

No installation, build tools, or additional dependencies are required.

## How to Play

1. Click on a block piece to select it.
2. Move your cursor over the board to preview its placement.
3. Green cells indicate a valid position, while red cells indicate an invalid one.
4. Click on a valid location to place the block.
5. Complete rows or columns to clear them and earn points.
6. Once all three pieces are used, a new set of pieces is generated.
7. The game ends when no remaining piece can be placed on the board.
